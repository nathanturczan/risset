# Risset Rhythm Generator - Max for Live MIDI Tool

A MIDI Generator for Ableton Live 12 that creates polyrhythmic Risset rhythm patterns.

## Installation

1. Download `risset.amxd`
2. Save it to: `User Library > MIDI Tools > Max Generators`
3. Restart Ableton if it's already open

## Usage

1. Create or select a MIDI clip
2. Open the MIDI Tools panel (wand icon)
3. Select "risset" from the Generators dropdown
4. Adjust parameters and click Generate

## Parameters

| Parameter | Description |
|-----------|-------------|
| Direction | Accel (speeds up) or Decel (slows down) |
| Mode | Arc (2 metabars, complete cycle) or Ramp (1 metabar, transition) |
| Ratio | Tempo ratio between layers (e.g., 2:3) |
| High/Low Pitch | MIDI notes for the two crossfading layers |
| Velocity Response | Shapes the crossfade curve (Punch to Gentle) |
